# Grunt ( Task Runner )

## To Install Globally
		npm install -g grunt-cli
    
## Create Simple web template
  html,css,js
 
## Create package.json
  npm init
  
## To add grunt plugin
  npm install --save <plugin_name>
  
## To Run Grunt Application
  grunt
  
  ```
    To Run this repository just run below command
    npm install -g grunt-cli
    npm install
    grunt
 ```
